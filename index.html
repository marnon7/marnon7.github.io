OS X下安装jupyter并且解决zsh: command not found: jupyter

Posted on 2020-08-01 Edited on 2020-08-03

安装python 和 pip 就不多说了，不会的童鞋，看后面的内容也意义不大，直接进入正题：

  1. 安装jupyter

    pip install jupyter

  2. 正确的话会提示安装成功

  3. 启动

    jupyter notebook

  • 然鹅，此时问题出现了:

    “zsh: command not found: jupyter”

  • 不要慌，分析一下：

    pip3 show jupyter

    返回

看上去是傻逼的自带python环境问题，不纠结这个，加个环境变量试试；
根据提示：

Location: /Users/yourusername/Library/Python/3.7/lib/python/site-packages

处理一下成为bin的路径

export PATH=/Users/**/Library/Python/3.7/bin:$PATH

加到zsh的环境变量里

vim .zshrc

source一下生效

source .zshrc

等灯等灯 okk啦

社交行业典型欺诈场景分析

Posted on 2019-10-22 Edited on 2019-10-30

Fraud Scenario Analysis of Social Platform

最近在整理互联网行业典型的欺诈的场景，首先会针对社交和电商行业做一些分析，接着或许会挑一些最近感兴趣的航旅类的行业。

首先抽象出来社交行业的一些核心业务环节：注册，登陆，加好友，UGC；评论/点赞；发消息，交易（转账，红包，打赏，刷礼物等）。将 UGC内容的发布和评论及私信放到一起来说，并不是这些业务重要或者风险低，正如我们知道的这些是业务社交平台非常重要的点，也是欺诈问题的重灾区。

注册

  • 虚假注册
  • 恶意批量注册
  • 推广红利滥用:裂变/师徒拉新

登陆

  • 账号盗用
  • 撞库攻击
  • 异常登陆检测

加好友

  • 虚假好友请求
  • 骚扰/广告

UGC;评论/点赞;发消息

  • 涉 “黄,赌,毒”
  • 垃圾消息
  • 骚扰/广告
  • 批量点赞
  • 推广红利滥用: 赚取活跃积分
  • 恶意爬虫

交易(转账/红包/打赏)

  • 红包赌博
  • 洗钱
  • 支付接口套现

凯文·凯利《必然》

Posted on 2017-07-07 Edited on 2019-09-29 In 商业

这是之前给公司整理的图书推荐语，后来种种原因好像没有用上，就贴上来吧，好吧，是因为好久没写东西了。

作者简介

凯文•凯利（1952~），绰号KK，世界著名科技杂志《连线》创始主编，“网络文化”（Cyberculture）的发言人和观察者，第一届世界黑客大会发起者。1994年，其预言未来科技发展的传奇著作《失控》横空出世，十几年后，文中所指相继出现，2010年，《失控》中文版问世，即在中国互联网界和科技媒体引起轰动，从此，KK 的一言一行都成为互联网发展的“风向标”。在互联网创业大潮席卷各个领域的今天，当媒体聚焦在路演、IPO 以及科技公司市值时，KK 依旧冷静地游走在最新科技产品与人迹罕至的原始部落中，思考科技该如何发展以及人类将怎样依靠科技，从而更加幸福地生活。也许只有这样，才可以预见到那些能够真正改变我们生活的伟大发明。KK 让我们相信，科技带给我们的未来，终将是一个充满温暖、人性与自由的“进托邦（Protolia）”。

必然 The Inevitable

“这些力量并非命运，而是轨迹。它们提供的并不是我们将去往何方的预测。只是告诉我们，在不远的将来，我们会向哪些方向前行，必然而然。”
